In-App Message
Design Guide

Design system, live examples, copy constraints, and implementation notes for Rapsodo in-app messages delivered through Iterable.

1 Overview

In-app messages are the highest-intent touchpoint in our lifecycle stack. Unlike push or email, they reach users while they are already in the app with interaction rates of 25–40% — roughly 10x push and 20x email.

We support three formats, each with a specific role:

~35%
Modal

Centered card over dimmed overlay. High-stakes feature discovery and conversion prompts.

~12.5%
Banner

Floating notification bar. Gentle nudges and informational prompts that don't block the UI.

Rare
Fullscreen

Dark full-bleed takeover. Reserved for milestones, celebrations, and major feature reveals.

2 Design Principles

Eight upgrades that move our in-app messages from "Iterable default template" to intentional brand communications.

2. Barlow Typography

Loaded via @import. 24px/800 headlines, 14px/400 body. Brand typeface, not system fonts.

3. Generous Spacing

32px card padding (up from 24px). 22px icon-to-headline gap. 28px body-to-CTA. Premium feel through breathing room.

4. Pill-Shaped CTAs

border-radius: 999px, 16px vertical padding. Matches native iOS/Android button patterns.

5. Layered Shadows

Tight near shadow + diffused far shadow. Reads native, not as a floating div. 0 2px 8px + 0 16px 48px.

6. Dark Card Variant

Dark background (#1a1a1a) for milestones and conversion moments. White text on dark reads as premium.

7. Higher Body Text Contrast

#3f3f46 replaces #52525b. Better OLED readability. Small change, noticeable improvement.

8. Proper Dismiss Affordance

Circle X button in top-right corner instead of text-only "Not now" link. Clear, tappable, and visible without feeling pushy.

4 Copy Constraints

Element Max Length Rules Example
Title 40 chars Benefit-focused. Lead with what the user gains, not the feature name. "See How You're Improving"
Body 150 chars (100 ideal) 1–2 sentences max. Reinforce value. No jargon. "Session Insights shows your trends across every session. Spot patterns and level up."
Button 1–3 words Uppercase. Action verb. No "Learn More" — too passive. VIEW INSIGHTS
Dismiss Required Always include. Circle X button or "Not now" text link. Never hide the exit. Circle X or "Not now"

Personalization

Use Handlebars merge tags when possible. {{firstName}}, {{total_shots}}, {{session_count}}. Data-driven copy outperforms generic every time. "You hit {{total_shots}} shots this week" beats "Keep practicing."

5 Timing Rules

Trigger on behavioral events only

Never calendar-based. Messages fire on session_complete, feature_opened, milestone_reached, etc. The user's action determines the message, not a scheduled date.

Post-session = highest intent

Right after a practice session ends, the user is engaged and primed for a next action. This is the best time for modals pointing to Session Review, Insights, or Courses.

Pre-session = banner, not modal

If the user just opened the app and hasn't started yet, use a banner. Don't block them with a modal before they've done what they came to do.

Never interrupt mid-session

While a user is actively hitting shots, no in-app messages. Zero exceptions. Wait for session_complete or a natural pause state.

InApp is independent of email/push cadence

In-app messages don't count toward email or push frequency caps. They're a separate channel. A user can receive an email AND an in-app message on the same day without overcommunication — they're in completely different contexts.

6 Iterable Implementation

Template Editor

Paste the full HTML into Iterable's Side-by-Side editor. All CSS must be inline — no external stylesheets. No JavaScript execution.

Barlow Font Loading

Add this at the top of the template HTML. Adds ~100ms load time — worth it for modals and fullscreen. Skip for banners.

<style>
  @import url('https://fonts.googleapis.com/css2?family=Barlow:wght@600;700;800&display=swap');
</style>

Deep Links

CTA buttons link to specific app screens via URI scheme:

Practice rapsodo://practice

Courses rapsodo://courses

Dismiss Action

Both the circle X button and "Not now" text link to:

<a href="iterable://dismiss">Not now</a>

Handlebars Merge Tags

Iterable supports dynamic personalization:

{{firstName}} {{total_shots}} {{session_count}} {{last_session_date}} {{combine_score}} {{trial_days_remaining}}
